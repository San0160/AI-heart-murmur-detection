# ❤️ AI Heart Murmur Detection

A machine learning project that detects heart murmurs from audio recordings using signal processing and deep learning.

---

## 📌 Overview

This project builds an end-to-end pipeline for classifying heart sounds into **normal** or **murmur** using audio features.

---

## 📊 EDA & Preprocessing

* Visualized waveforms to understand heart sound patterns
* Standardized sampling rate and cleaned audio
* Reduced noise and normalized signals

---

## 🔍 Feature Extraction

* Extracted **MFCC (Mel-Frequency Cepstral Coefficients)**
* Aggregated features across time for model input

---

## 🤖 Model

* Trained using TensorFlow/Keras
* Input: MFCC features
* Output: Binary classification (Normal vs Murmur)

---

## 🚀 App

* Built with Streamlit
* Upload audio → extract features → predict murmur

---

## ⚙️ Tech Stack

Python, TensorFlow, Librosa, NumPy, Streamlit
